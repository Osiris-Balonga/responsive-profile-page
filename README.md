# Responsive Profile Page

Page profil responsive construite avec HTML5 et CSS vanilla.

## Fichiers

- `index.html`
- `style.css`
- `assets/`

## Contenu

- Header, navigation, contenu principal et footer
- Photo, nom, bio, competences, formation et reseaux sociaux
- Formulaire de contact avec labels
- Dark mode CSS-only
- Layout mobile first avec plusieurs breakpoints
- Validation HTML
